# Pizza-Order-Form
This project is a simple pizza ordering form built using HTML and CSS only. It allows customers to enter their personal details, choose pizza size, crust, sauce, toppings, delivery mode, and payment information. The from follows specific validation rules using HTML attributes to ensure correct user input. CSS styling is optional (but we decided to still add CSS to make the form more presentable), and no JavaScript is used.
